# Malabs Group Landing

Landing page estática de Malabs Group construida con Astro y Tailwind CSS.

## Branding

- El sitio usa el logo existente en `public/brand/malabs-logo-stacked.svg`.
- También está disponible `public/brand/malabs-logo-mark.svg` para usos alternos.
- El favicon placeholder está en `public/favicon.svg`.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloudflare Pages

- Build command: `npm run build`
- Build output directory: `dist`
